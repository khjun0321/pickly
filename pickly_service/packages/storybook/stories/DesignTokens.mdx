## Typography

### Font Families

- **Primary**: Pretendard Variable, Pretendard, -apple-system, BlinkMacSystemFont, system-ui, Roboto, sans-serif
- **Mono**: SF Mono, Monaco, Inconsolata, Roboto Mono, monospace

### Font Sizes

| Token | Size | Usage |
|-------|------|-------|
| `--font-size-xs` | 12px | 작은 보조 텍스트 |
| `--font-size-sm` | 14px | 일반 텍스트, 버튼 |
| `--font-size-base` | 16px | 기본 본문 텍스트 |
| `--font-size-lg` | 18px | 큰 본문 텍스트 |
| `--font-size-xl` | 20px | 작은 제목 |
| `--font-size-2xl` | 24px | 중간 제목 |
| `--font-size-3xl` | 30px | 큰 제목 |
| `--font-size-4xl` | 36px | 매우 큰 제목 |

### Font Weights

| Token | Weight | Usage |
|-------|--------|-------|
| `--font-weight-light` | 300 | 가벼운 텍스트 |
| `--font-weight-normal` | 400 | 일반 텍스트 |
| `--font-weight-medium` | 500 | 중간 강조 |
| `--font-weight-semibold` | 600 | 강조 텍스트 |
| `--font-weight-bold` | 700 | 볼드 텍스트 |

### Line Heights

| Token | Value | Usage |
|-------|-------|-------|
| `--line-height-tight` | 1.25 | 제목용 |
| `--line-height-normal` | 1.5 | 일반 텍스트 |
| `--line-height-relaxed` | 1.75 | 긴 글 읽기용 |

## Spacing

8px 기반의 간격 시스템을 사용합니다.

| Token | Size | Usage |
|-------|------|-------|
| `--spacing-1` | 4px | 아주 작은 간격 |
| `--spacing-2` | 8px | 작은 간격 |
| `--spacing-3` | 12px | 기본 간격 |
| `--spacing-4` | 16px | 일반 간격 |
| `--spacing-5` | 20px | 중간 간격 |
| `--spacing-6` | 24px | 큰 간격 |
| `--spacing-8` | 32px | 더 큰 간격 |
| `--spacing-10` | 40px | 매우 큰 간격 |
| `--spacing-12` | 48px | 섹션 간격 |
| `--spacing-16` | 64px | 큰 섹션 간격 |
| `--spacing-20` | 80px | 매우 큰 섹션 간격 |

## Border Radius

| Token | Size | Usage |
|-------|------|-------|
| `--border-radius-none` | 0px | 각진 모서리 |

| `--border-radius-base` | 4px | 기본 둥글기 |
| `--border-radius-md` | 6px | 중간 둥글기 |
| `--border-radius-lg` | 8px | 큰 둥글기 |

| `--border-radius-2xl` | 16px | 아주 큰 둥글기 |
| `--border-radius-full` | 50% | 완전 원형 |

## Shadows

| Token | Shadow | Usage |
|-------|--------|-------|
| `--shadow-xs` | 0 1px 2px rgba(16, 24, 40, 0.05) | 아주 작은 그림자 |
| `--shadow-sm` | 0 1px 3px rgba(16, 24, 40, 0.1) | 작은 그림자 |
| `--shadow-md` | 0 4px 8px rgba(16, 24, 40, 0.1) | 중간 그림자 |
| `--shadow-lg` | 0 12px 16px rgba(16, 24, 40, 0.1) | 큰 그림자 |
| `--shadow-xl` | 0 20px 24px rgba(16, 24, 40, 0.1) | 매우 큰 그림자 |
| `--shadow-2xl` | 0 24px 48px rgba(16, 24, 40, 0.18) | 아주 큰 그림자 |

## 사용 예시

### CSS에서 사용

```css
.my-button {
  background-color: var(--color-primary-500);
  color: var(--color-white);
  font-family: var(--font-family-primary);
  font-size: var(--font-size-sm);
  font-weight: var(--font-weight-medium);
  padding: var(--spacing-3) var(--spacing-4);
  border-radius: var(--border-radius-md);
  box-shadow: var(--shadow-sm);
}
```

### JavaScript/TypeScript에서 사용

```javascript
import { tokens } from '@pickly/design-tokens';

const buttonStyles = {
  backgroundColor: tokens.colors.primary[500].value,
  color: tokens.colors.white.value,
  fontFamily: tokens.typography.fontFamily.primary.value,
  fontSize: tokens.typography.fontSize.sm.value,
  fontWeight: tokens.typography.fontWeight.medium.value,
  padding: `${tokens.spacing[3].value} ${tokens.spacing[4].value}`,
  borderRadius: tokens.borderRadius.md.value,
  boxShadow: tokens.shadows.sm.value,
};
```
